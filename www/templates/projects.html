Improved Initiative

Improved Initiative is an Android app created to help Dungeon Masters manage turn order in pen-and-paper tabletop roleplaying games like Dungeons and Dragons. The app is written entirely in Java and was develped using Android Studio. SQLite was used as backend local storage for persistent character data.

Scoundrel

Scoundrel is a terminal based "rogue-like" RPG for Linux built in C using the ncurses library. In its current state, the player can traverse a static dungeon loaded from a file, slay monsters, and collect loot. Random dungeon generation is planned, along with a level-up system and more in-depth equipment management.

Automated Data Processing and Analysis

Processing large collections of scientific data by hand is both tedious and error prone, and introduces an additional source of bias. This set of scripts written in Python and R allow for automated data cleaning and analysis, facilitating fast, accurate statistical analyses and generation of graphs and figures.

PICAXE Robot

This small robot was programmed in BASIC on a PICAXE board, and is capable of autonomous roaming and obstacle avoidance. Additionally, the robot's automous roaming features may be overriden, allowing the robot to be controlled via an NES controller.
