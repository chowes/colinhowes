Personal Website

This simple (and hopefully appealing) website was built using basic HTML and CSS along with the AngularJS and Bootstrap frameworks. The snappy load times you may have noticed when navigating between pages were accomplished by using the "single page" model made possible (or at least easier) by some cool features in AngularJS. You can also also open this site on a mobile phone (or just change the size of your browser window) if you would like to see the nice responsive UI I implemented through some neat Bootstrap style features.
